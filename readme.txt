1. "npm install" ile paketleri yükleyin.
2. "npm start" ile index.js çalıştırılır.
3. Uygulama ".env" klasöründeki linkten MongoDB localhosta bağlanmaktadır.

NOTLAR
- Deneme için oluşturduğum MongoDB koleksiyonlarını "mongo_collections" klasörüne export ettim.
- Yine kullandığım postman sorgularını "Lidyverse.postman_collection.json" dosyasında bulabilirsiniz.
